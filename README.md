Description
------------------------------------------------------------------

A small website in php to manage courses submissions and rank them.


Environment
------------------------------------------------------------------

The following is the format that the `.env` file need to take to let the website link to the database:

~~~ini
DB_HOST=XXX
DB_USER=XXX
DB_PASSWORD=XXX
DB_NAME=XXX
~~~
